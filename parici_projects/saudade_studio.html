Че это?

Saudade Studio - команда людей, которая создает модпаки и моды для Minecraft на энтузиазме.

Что я там делал?

В этой студии я занимался созданием модов для модпака Desiderium. В силу ещё плохого понимания темы программирования, а уж особенно объектно-ориентированного Java мне приходилось выкручиваться, поэтому моды создавались в MCreator (программа, работающая по принципу генерации кода по шаблону и дальнейшей вставки его в итоговый проект и созданная для тех, кто не умеет писать код, но хочет создавать моды для Minecraft).
